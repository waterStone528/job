帮助说明

如何投资

融资发布后，可投时间为当天。当天23:59分后自动停止。

每月还款日固定为15号。例如六期还款，每15号还总融资本息的1/6。

如何融资

融资服务费率为实际融资金额的0.12%。 融多少算多少，在投资收款时扣除。

投资转让

请使用智能帮助系统，点击右上角帮助图标再点你想要详细了解的信息内容。

关键说明

日利率 为月利率/30。

系统要求

受支持的软件环境，以下任意一种：

  • Windows XP Pro：IE8
  • Windows Vista：IE9/IE10/IE11
  • windows 7 Pro/Bin/bas：IE9/IE10/IE11
  • windows 8 RT：IE9/IE10/IE11
  • windows 8.1 RT：IE10/IE11
  • windows 8 Pro/MCE：IE9/IE10/IE11
  • windows 8.1 Pro/MCE：IE10/IE11

支持的硬件输入设备：

  • 支持触摸屏操作：我们平台能适应触摸屏操作并加入了专项优化。通过测试的有Surface，大屏Windows手机等。
  • 己知各种类型的二键、三键鼠标。

支持的显示设备：

  • 各种形式的分辨率大于等于1366*768的显示屏。

以上操作系统与浏览器己测试并受支持。为了获得最佳运行效果，建议使用要求的环境。
