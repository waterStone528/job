网站资产交易平台服务特定条款

一、基本定义。

1.1 本交易平台所指的资产/出售标的是指法人及其他经济组织、自然人所拥有的具有实物形态的资产，主要包括房地产、机械设备、机动车及其他交通运输工具、物品、存货等资产。

1.2 出售方：指对房地产、机械设备、机动车及其他交通运输工具、物品、存货等资产享有所有权或有权加以处置并意向销售的一方。

1.3 购买方：指有意向购买上述出售标的的一方。

1.4 该资产交易的出售方和意向购买方可以通过在本平台的预约等方式实现交易。

二、资产的出售

2.1 出售方转让资产，应当具有出售该出售标的的主体资格，保证该出售标的可以合法转让，并在必要时应依照相关法律、法规和政策的规定履行内部决策、资产评估等相关程序，如出售方为国有及国有控股企业的，还应提交有权批准机构同意转让的批复或决议及资产评估结论等。

2.2 出售标的为土地使用权、房地产的，权属证明文件包括国有土地使用权证、房屋所有权证或具有法律效力的其他权属证明文件等。

出售标的为机动车的，车辆应在年检期内且无欠缴的各项费用（包括但不限于违章罚款），权属证明文件包括机动车登记注册证、机动车行驶证、购置税完税凭证、保险单等。

2.3 出售方应对所提交材料的真实性、完整性、有效性负责。

2.4 出售方转让已对外租赁房屋且承租人有优先购买权的，以及转让共有物且其他共有人有优先购买权的，应当依法履行相应的程序。

2.5 出售方可自主在本网站平台进行信息发布。详请参照《平台使用手册》。

2.6 意向购买方可自主筛选出售方发布的资产项目进行购买。详请参照《平台使用手册》。

2.7 网站平台上发布的信息包括交易双方、标的质量和价值等，并未经本网站最终进行法律/财务上的核实，交易双方应依其独立判断做出决策。双方据此进行交易的，产生的风险均由出售方/购买方自行承担，且无权据此向网站平台及其所有者和相关方提出任何法律主张。在交易过程中，出售方和购买方之间发生的任何纠纷，由纠纷各方自行解决，网站平台不承担任何责任。

3.1 信息发布期间，如出售方与购买方确立交易意向的，则由交易双方自行签订资产出售合同；网站平台不具体介入该交易的内容及交易的方式等，也不对交易行为承担责任。

四、结算交易资金及产权的移转

4.1 资产交易各方应通过约定的资金支付方式进行交易资金的结算，网站平台不予以介入/干涉，也不承担结算责任。

实物资产交易资金指交易价款，以人民币为结算货币。

交易争议经协商或者调解未能解决的，当事人可以向仲裁机构申请仲裁或向人民法院提起诉讼，该争议与交易平台无涉。
