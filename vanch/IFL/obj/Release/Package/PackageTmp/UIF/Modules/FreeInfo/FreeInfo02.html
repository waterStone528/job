岗位招聘

市场企划总监/经理

岗位职责

  • 负责公司品牌发展战略，全面制定线上品牌战略规划及实施
  • 公司网站平台对外视觉把控及传播，对外宣传的推广方案制定
  • 依据公司品牌定位，制定产品拍摄及推广方案
  • 依据品牌特点，做好产品相关包装、元素等的设计
  • 负责各类企划活动文案的起草设计，执行企划活动项目

任职要求

  • 本科以上学历，设计相关专业毕业
  • 5年以上企划相关工作经验，2年以上主管工作经验
  • 具备一定媒体关系维护经验
  • 性格开朗，擅长沟通，具备较强的规划总结能力
  • 积极上进，责任感强，较强的团队管理能力

风控总监/经理

岗位职责

  • 负责制定公司项目风险控制的指导原则，不断检查审批环节和审查内容，并提出完善建议
  • 负责对担保项目进行综合风险评估，填写《风险控制评估报告》，明确风险防控措施
  • 根据公司担保业务的流程，做好“事前介入，事中参与，事后复核”的工作即在担保业务初步受理阶段，对反担保措施提供法律建议
  • 在担保业务调查阶段，参与相关合同的起草、谈判、修改、签订、公证、登记等过程，对客户提供的担保资料进行真实性和合法性的审核工作
  • 在放款阶段，做好相关程序和手续的复核防止错漏
  • 在保后阶段，对担保客户贷款的日利率及时偿还进行催促
  • 协助公司领导正确执行国家的法律、法规，对公司的重大经营决策活动提供法律意见
  • 根据公司的安排，参与公司的资本运营，特别是收购、租赁、招标、投标等重大经济活动，草拟相关合同文件，参与相关的谈判，提出相关的风险控制意见
  • 配合公司的有关安排，在公司内普及相关知识，增强职员的风险意识
  • 关注相关市场政策调整和变化，制定应对调整和变化的相关办法
  • 负责办理公司领导交付的其他任务

任职要求

  • 金融、经济、投资、法律或财务管理相关专业，大专以上学历
  • 银行或大型资产管理企业五年以上工作经验，同等岗位三年以上工作经验
  • 熟悉投资行业资产经营管理工作，有丰富的实战经验
  • 熟悉收购、兼并、上市、退出的工作流程和丰富的经验
  • 很强的判断、沟通、交往能力及组织协调能力
  • 有银行担保行业或审计、经营信贷风险管理经验优先入取
  • 年龄28-35岁

资金部经理

岗位职责

  • 资源整合：负责金融机构及投资公司资源拓展
  • 资源维护：负责资金资源的维护工作
  • 资源对接：根据信贷部提供的项目对接相应资金资源
  • 贷后维护：跟踪已放贷项目的相关情况
  • 资料调整：根据实际情况对信贷部提供的项目资料进行分析研究，并建议改进方案
  • 部门管理：管理本部门成员，提高部门工作效益

任职要求：

  • 基本要求： 女性，28-38岁 良好的形象素质
  • 学历要求：大专以上学历，金融、经济、会计相关专业
  • 经验要求：3年以上管理岗位经验，熟悉融资业务及融资渠道，熟悉国际、国内金融产品，精通资金管控流程
  • 能力要求：有较强高端社交能力有较好管理和培训能力
  • 品性要求：勤奋上进，耐心沉稳，有忠诚度

.net程序员（开发工程师）

岗位职责：

  • 在上级的领导和监督下定期完成量化的工作要求
  • 能独立处理和解决所负责的任务
  • 根据开发进度和任务分配，完成相应模块软件的开发、编程任务
  • 进行程序单元、功能的测试，查出软件存在的缺陷并保证其质量
  • 进行编制项目文档和质量记录的工作

任职要求：

  • 有2年以上asp.net编程经验，能够熟练使用C#，熟悉.net框架
  • 熟悉B/S结构，对web有深刻的了解，熟悉web的工作原理能熟练使用Web开发相关的HTML，CSS，Javascript
  • 善用AJAX技术，了解XML，WebServices
  • 熟悉主流数据库(sql2000/2005/2008/Oracle)，能熟练编写SQL语句、存储过程
  • 做事严谨踏实，责任心强，条理清楚，善于学习总结，能够独立分析问题并解决问题，有良好的团队合作精神和沟通协调能力

信贷经理

岗位职责：

  • 根据公司战略规划负责信贷业务
  • 根据相应的渠道开拓客户，并维护好客户关系，做好信息录入、整理工作
  • 负责接待及回复客户关于贷款问题的咨询
  • 根据公司要求严格审核信贷业务信息资料，融资方财务状况及贷款能力进行评估，控制信贷风险
  • 根据项目需求对接相应资金，包括自有资金和外部资金
  • 负责回收贷款和控制贷款风险
  • 支持p2p平台信贷业务
  • 负责服务、维护客户，为客户提供优质的贷前、贷中及贷后服务

任职要求

  • 金融或相关专业，大专以上学历，形象气质佳
  • 三年以上金融管理岗位工作经历，熟悉信贷业务及相关政策
  • 具有扎实的经济、金融、投资等领域的相关理论知识
  • 强有力的沟通能力，良好的公众演讲能力和社交谈判能力

审计经理

岗位职责：

  • 负责公司业务项目的财务审计具体实施工作，包括审计计划拟定，组织实施完成
  • 通过审计，找出财务管理中存在问题和主要风险，并提出有效的改善建议
  • 后续审计跟踪，对审计发现的问题改善情况进行跟踪，必要时进行跟踪审计
  • 根据公司要求严格审核信贷业务信息资料，融资方财务状况及贷款能力进行评估，控制信贷风险
  • 定期梳理和评估财务审计工作流程，并据此进行完善和优化
  • 实事求是，客观公正地提供相关信息，撰写审计报告或单项报告送审稿
  • 根据工作中发现的管理漏洞及经营风险提出报相关管理建议，对经营中易出现问题的环节进行预警和提示

任职要求

  • 25至35周岁，财务、金融类专业本科及以上学历，具有审计师、国际注册内审师、中级会计师、注册会计师其中职称
  • 3年以上的会计师事务所审计或风险管理工作经验
  • 具备投资型企业审计工作经验，熟悉审计的工作流程和管理
  • 熟练操作财务软件，熟悉国家审计法律法规，有较好的沟通和协调能力和文字能力
  • 能够承受工作压力，性格外向，善于沟通，执行力强，勇于创新，有良好的职业操守，正直、诚信、公正、廉洁，具有现代成本控制理念和审计监察的强硬作风

法务经理

岗位职责：

  • 协助公司领导正确执行国家法律法规，对公司的重大经营决策进行法律论证，出具法律意见
  • 负责公司合同文本的制定、修订，参与公司重大合同的谈判、签订，对各类合同的履行进行监督
  • 负责拟定、审核及完善公司的各项规章制度
  • 负责处理公司对外业务中涉及到的法律事务，并对业务对象提供法律支持
  • 对公司运营中的各种潜在法律风险进行梳理、识别，并提出规避、化解方案
  • 代表公司参与涉诉事务的协商、调解、诉讼与仲裁活动
  • 对侵犯公司利益的事项进行调查、处理，依法追究相关人员的法律责任
  • 与司法机关及有关政府部门保持沟通，为公司创造良好的司法环境
  • 负责商标专利等公司知识产权的申请、维护
  • 负责企业内部的法律宣传、教育及培训

任职要求

  • 专科及以上学历，法律专业
  • 5年以上法务管理或相关法律工作经历，有金融行业从业经验或律师执业资格证的优先
  • 有较强的法律逻辑思维能力、判断与决策能力、商务谈判能力、人际沟通及语言表达能力
  • 良好的沟通、组织和协调能力，成熟稳重，思维敏捷，工作踏实，能全面管理公司法律相关工作
  • 有较强的法律风险识别能力
